簡単設定

Firebase CLIを使用した簡単なコマンドでワンクリックデプロイが可能

設定の簡単さ

Firebase Hostingは、開発者が最短時間でWebサイトをデプロイできるよう設計されています。 複雑な設定は不要で、数分でグローバル配信を開始できます。

⚡ 3ステップデプロイ

Firebase CLIをインストールし、プロジェクトを初期化して、デプロイするだけ。 わずか3つのコマンドで、あなたのWebサイトが世界中からアクセス可能になります。

セットアップステップ

1

Firebase CLI のインストール

Node.js環境でFirebase CLIをグローバルインストール

npm install -g firebase-tools
2

Firebaseにログイン

Googleアカウントでログインし、認証を完了

firebase login
3

プロジェクト初期化

Hostingを選択し、設定ファイルを生成

firebase init hosting
4

デプロイ実行

ファイルをアップロードし、即座に本番公開

firebase deploy

簡単設定の特徴

コマンドライン操作

GUI不要のシンプルなコマンドライン操作。 開発者に優しいCLIツールで効率的な作業が可能です。

  • 直感的なコマンド構造
  • 詳細なヘルプ機能
  • エラーメッセージが分かりやすい
  • 自動補完サポート

自動設定生成

firebase.jsonとfirebaserc設定ファイルが自動生成。 プロジェクト構造に応じた最適な設定を提案します。

  • インタラクティブ設定
  • デフォルト値の自動設定
  • カスタマイズ可能な設定
  • 設定の検証機能

継続的デプロイ

GitHub Actionsとの統合により、 コードプッシュ時の自動デプロイを簡単設定。

  • CI/CD パイプライン統合
  • 自動テスト実行
  • プレビューURL生成
  • ロールバック機能

フレームワーク統合

React、Vue、Angular、Next.jsなど、 主要フレームワークとの統合設定をサポート。

  • フレームワーク自動検出
  • ビルド設定の最適化
  • SPA ルーティング対応
  • SSG/SSR サポート
