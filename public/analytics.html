詳細分析

リアルタイムの使用状況分析とパフォーマンス監視機能で深いインサイトを提供

分析機能概要

Firebase Hostingの分析機能は、Webサイトのパフォーマンスとユーザー行動を 詳細に追跡し、データドリブンな意思決定をサポートします。

📊 リアルタイム分析

Google Analyticsとの完全統合により、訪問者数、ページビュー、 ユーザー行動をリアルタイムで監視できます。

主要分析機能

ユーザー行動分析

訪問者の行動パターンを詳細に分析。 ページ滞在時間、離脱率、コンバージョン経路を可視化します。

  • ページ別滞在時間
  • ユーザーフロー分析
  • 離脱ページ特定
  • コンバージョン追跡

パフォーマンス監視

Core Web Vitalsを含む詳細なパフォーマンス指標で、 ユーザー体験の質を継続的に監視します。

  • ページ読み込み速度
  • Core Web Vitals
  • ネットワーク分析
  • エラー率監視

地域別分析

世界中のユーザーアクセスを地域別に分析。 CDN効果やローカライゼーションの効果を測定できます。

  • 国・地域別アクセス
  • 言語設定分析
  • 時間帯別トラフィック
  • 地域別パフォーマンス

カスタムアラート

重要な指標の変化を自動検知し、 即座にアラートを送信します。

  • トラフィック急増検知
  • エラー率上昇警告
  • パフォーマンス低下通知
  • カスタム閾値設定

データエクスポート

分析データを様々な形式でエクスポート。 外部ツールとの連携や詳細分析が可能です。

  • CSV/JSON エクスポート
  • API 経由でのデータ取得
  • 定期レポート生成
  • BigQuery 連携

プライバシー準拠

GDPR、CCPA等のプライバシー法規に準拠した データ収集と分析を実現します。

  • プライバシー設定
  • データ匿名化
  • 同意管理
  • データ保持期間設定

実装ガイド

1

Google Analytics設定

Google Analytics 4プロパティを作成し、測定IDを取得

<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
2

Firebase Analytics有効化

Firebase ConsoleでAnalyticsを有効化

firebase init analytics
3

Performance Monitoring設定

パフォーマンス監視を有効にして詳細データを収集

import { getPerformance } from 'firebase/performance';
4

カスタムイベント設定

ビジネス固有のイベントを追跡設定

gtag('event', 'custom_event', { 'custom_parameter': 'value' });
